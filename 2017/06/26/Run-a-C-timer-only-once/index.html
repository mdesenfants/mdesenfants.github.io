Run a C# timer only once

Published on Monday, June 26th 2017 at 10:53
Last updated on Monday, June 26th 2017 at 11:10

Question

How do I run a C# timer only once?

Answer

Use Task.Delay(TimeSpan).ContinueWith(t => code).

Task.Delay(Timespan.FromSeconds(10))
    .ContinueWith(_ => Console.WriteLine("That's time."));

Details

I originally posted this on StackOverflow when I realized most people were still suggesting start-and-stop timer techniques intended for desktop apps.

You can optionally add a cancellation token if you need to stop the timer before executing the anonymous function.
