Base 64 from C# to Javascript

2017-06-26

Question

How do I serialize bytes to base 64 from C# for a web client?

Answer

There are two ways, depending on the situation.

To be sent as a header or a URL parameter

Server encode

Client decode

To be sent in the response body

Server encode

Client decode

Details

The two situations described above can pop up in different ways. For example, say you want to just send some sort of blob to the client. You can easily send this in the response body and have javascript parse the response without an issue with the second (more familiar) solution in C#.

However, the first solution crops up when you need to send something like a JWT or other Bearer token across the request. In this case, all strings must be URL-safe. OWIN provides the Base64UrlTextEncoder.aspx) for this purpose.
